    성결대학교 컴퓨터공학과 팀 HELPER 졸업작품인 학과 전용 카카오 챗봇 프로젝트
# Prerequisites
    nodeJS V16
    
    expressJS V4.17.2
    
    firebase V10.2.1
    
    eslint V8.13.0
    
    mocha V9.1.4
    
    Heroku Scheduler & Advanced Scheduler
# Get start
    > cd functions
    > npm install 
    > npm run config
    > npm start
